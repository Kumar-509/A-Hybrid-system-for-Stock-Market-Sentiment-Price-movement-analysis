# A-Hybrid-system-for-Stock-Market-Sentiment-Price-movement-analysis
Hybrid Stock Price Prediction System using LSTM and NLP to forecast future OHLC values. Combines historical stock data with sentiment analysis from news and social media. Features a 1-week forecast dashboard, interactive date selection, and visual insights for smarter investment decisions.

🧠 Features

📊 Predicts Open, High, Low, Close (OHLC) values
📅 1-week stock forecast
🧾 Sentiment analysis using NLP
🔗 Hybrid model (LSTM + Sentiment)
📈 Trend visualization
🖥️ Interactive web dashboard
📌 Date-based prediction

🛠️ Tech Stack

Language: Python
ML/DL: TensorFlow, Keras / PyTorch
NLP: NLTK, VADER, TextBlob
Data: Pandas, NumPy
Visualization: Matplotlib, Seaborn
Backend: Flask / Django
Frontend: HTML, CSS, Bootstrap

🏗️ Project Workflow

Data Collection → Data Preprocessing → 
Sentiment Analysis → LSTM Model → 
Hybrid Integration → Prediction → UI Dashboard

📂 Project Structure
├── data/                 # Historical stock datasets
├── models/               # Trained LSTM models
├── sentiment/            # NLP & sentiment analysis
├── static/               # CSS, JS files
├── templates/            # HTML files
├── app.py                # Main backend application
├── utils.py              # Helper functions
├── requirements.txt      # Dependencies
└── README.md

⚙️ Installation
# Clone the repository
git clone https://github.com/Kumar-509/A-Hybrid-system-for-Stock-Market-Sentiment-Price-movement-analysis.git

# Navigate to project folder
cd A-Hybrid-system-for-Stock-Market-Sentiment-Price-movement-analysis

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
